# 개요

> 블로그용 보일러 플레이트

# 주요 특징
* Svelte-kit을 이용한 블로그 구현
* Static site generation로 빌드
* Svelte용 mdx 적용
* Github Page에 배포할 수 있도록 세팅

## 빌드
> yarn을 사용해 주세요 :)

```bash
# 의존성 모듈 설치
yarn install

# 앱 빌드
yarn build

# 로컬개발 서버 실행
yarn dev -- --open
```

## 배포
> root에 있는 gh-pages.js를 먼저 설정해주세요 :)

```bash
# 배포
yarn deploy
```
